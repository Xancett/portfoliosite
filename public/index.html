About me

Hangman Game

Info: Hangman game designed to focus on nodejs as well as a NoSQL database. The game allows you to play through a round with a randomly chosen word from my database. The word is never stored client-side to prevent cheating. When the game is over the win or loss is recorded and stored in the database as well. The stats and summary page is for viewing interesting stats pulled from the database about the games that have been played.

Tech: HTML, CSS, NodeJS, Mongodb

Directory Email Scrapper

Info: This is a script I wrote in my free time for a previous job. This script takes the first and last name of someone and uses the BeautifulSoup4 api to traverse the directory. Once the employee is found, their name and email is saved in a text file for use later. I wrote this to cut down on how much work was required before I could send out an email to multiple people in a list.
